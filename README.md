# Personal Professional Portfolio 🚀

A clean, responsive, and professional personal portfolio website built as **Task 1** for the **Future Interns** Full Stack Web Development program.

## 🌍 Live Demo
[Click here to view my live portfolio](https://sandeepbaratam.github.io/FUTURE_FS_01/) 

## 🎯 Objective
The goal of this project is to create a digital resume that showcases my technical skills, projects, achievements, and provides a clear way for recruiters and peers to contact me. This mirrors how real developers present themselves in the industry.

## ✨ Features
* **Modern UI/UX**: Clean layout with a professional "Slate and Navy Blue" color palette.
* **Fully Responsive**: Optimized for both desktop and mobile viewing.
* **Interactive Elements**: Smooth scrolling navigation and interactive hover effects on project cards.
* **Dedicated Sections**: Includes About Me, Technical Skills, Featured Projects, Achievements, Certifications, and Contact info.

## 🛠️ Tech Stack
* **HTML5**: Semantic structure.
* **CSS3**: Custom styling, CSS Grid/Flexbox layouts, and modern UI variables.
* **JavaScript**: Vanilla JS for smooth scrolling behavior.
